## Description

Back-end for ABC Shop
พัฒนาโดย Framework Nest.js และ MySQL Database 

## Requirement

```bash
$ Node.js Version 16+
$ MySQL Database Server (Run on port 3306)
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Note

```bash
- Application รันบน port 3000
- รายละเอียดการตั้งค่า Database อยู่ที่ไฟล์ app.module.ts
```
